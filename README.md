# Full Stack Assessment Submission

Duration: 1.5 Hours

## Instructions

1. Fork this repository.
2. Clone your forked repository to your laptop.
3. Create a folder using the following format:

firstname-lastname-collegename

Example:
rahul-sharma-nirmauni

4. Add your assessment code inside that folder.
5. Commit and push your code to your forked repository.
6. Create a Pull Request to submit your solution.

## Pull Request Title Format

Submission - Your Name

Example:
Submission - Rahul Sharma

## Pull Request Description

In the description section include:

Date: (today’s date)
College: (Nirma Uni)
Branch: (IT / CP / etc.)

Example:

Date: 12 March 2026
College: Nirma University
Branch: IT

## Example Folder Structure

rahul-sharma-nirmauni/
frontend/
backend/

## Submission

After creating the Pull Request, your submission will appear in the **Pull Requests** section of this repository.
